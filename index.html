Further reading

Create a repository on GitHub

1. Go to GitHub and click New repository.

2. Choose a name and click Create repository.

Connect your local repo to GitHub

From your project folder:

  • git remote add origin https://github.com/<you>/<repo>.git
  • git push -u origin main

Common notes

  • The name origin is just a convention for the default remote.
  • You may need to sign in / set up SSH keys or a token depending on your GitHub settings.
